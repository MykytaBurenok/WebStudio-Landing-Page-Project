# WebStudio Landing Page | Responsive HTML & CSS Website

A responsive multi-section landing page built with HTML5 and CSS3.  
This project was created to practice semantic markup, responsive layouts, and clean UI structure while focusing on maintainable styling and modern frontend layout techniques.

The goal was not only to recreate a static design, but also to build a polished, mobile-friendly interface with consistent spacing, adaptive sections, and clean responsive behavior across different screen sizes.

## Features

- Responsive layout for desktop, tablet, and mobile
- Semantic HTML structure
- Mobile navigation menu
- Hero section with call-to-action
- Content sections with cards and images
- Footer with contact and social links
- Flexible layout built with Flexbox and CSS Grid
- Clean and maintainable CSS structure

## Challenges

One challenge was adapting large desktop sections into clean mobile layouts while maintaining spacing consistency and visual hierarchy across different screen sizes.

Another focus area was creating reusable layout patterns that kept the design visually balanced without relying on fixed positioning or hardcoded dimensions.
